## shuffleAR()

EN: Randomize the order of the elements in the array

TR: Dizideki elemanların sırasını rastgele seçmek

### Usage / Kullanımı

```java-script
shuffleAR(array)
```
array: javascript array

### Return / Dönen Değer

```
array: javascript array
```

### Example / Örnek

```java-script
var arr = ['a','b','c','d','e','f'];
arr = shuffleAR(arr);
```
